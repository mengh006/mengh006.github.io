字重使用指南

情况二：独立字体名称

推荐使用情况一：统一管理更高效，代码更易维护

关键属性说明

font-display: swap

  • 作用：字体加载期间显示备用字体，避免页面空白
  • 推荐阅读：《减少字体加载影响的新方法》

unicode-range 优化

当你引入官方的 css 字体链接后，有些文件里的一个 unicode-range 属性引起了你的注意，这个属性的作用是规定不同的字符加载不同字符集文件。通俗讲我们会把一个字体分成多个字符集，然后按需加载。例如一个页面只有 500 个字，但是这 500 个字恰好都在一个字符集里面，那么其他的字符集就不需要加载，换句话说就不需要加载整个字体文件，因此节省了流量和优化了加载速度。

  • 原理：按需加载字符子集，减少流量消耗
  • 支持字体：思源黑体、小米字体、鸿蒙字体、华康金刚黑
  • 效果：500字页面可能只需加载单个字符集文件

字体格式选择

  • 本文统一使用 woff2 格式
  • 现代浏览器（Chromium内核）完美支持
  • 旧浏览器兼容方案参考：《关于字体格式》

结语

Windows 上微软雅黑仍有最佳显示效果（因特殊优化），但随着显示器分辨率提升，安装苹方等现代字体在 Windows 上也能呈现出色效果。字体选择应兼顾设计美感与技术实现，根据实际需求选择最佳方案。
